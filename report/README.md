# aliveKat/report/

latex files for the report and talk

- main.tex, bib.bib, myunsrt.bst compile the report (run main.tex) and must be run in this directory as to see all the figures
- talk.tex compiles the talk slides in beamer

```bash
    │   └── ...
    ├── README.md
    ├── main.tex
    ├── bib.bib
    ├── myunsrt.bst
    └── talk.tex
```
